# blue-team-simulations

This repository documents my hands-on Blue Team exercises focused on SOC 1 and SOC 2 analyst responsibilities. Through simulated attacks, security alerts, and incident response scenarios, I practice the workflows used to detect, investigate, document, and respond to security events in an enterprise environment.

The goal is to build practical experience in alert triage, log analysis, incident response, threat hunting, ticketing, and security reporting while developing repeatable procedures aligned with industry best practices.

> Learning security by doing the work—one alert, investigation, and incident report at a time.
